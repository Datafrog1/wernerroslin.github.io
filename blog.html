Site Makeover: From Scattered Code to Unified Architecture

July 8, 2025

Just pushed a major update that completely restructured how this site works under the hood. What started as separate HTML files with duplicated styles has evolved into a clean, maintainable architecture.

🔧 What Changed?

  • Unified CSS: Created a single styles.css that handles all global styles and dark mode
  • Shared JavaScript: Added scripts.js for common functionality across all pages
  • Dark Mode Toggle: Every page now has a consistent theme switcher (🔄 button in top-right)
  • Battleship Game: Properly centered and responsive with full dark mode support
  • Keyboard Shortcuts: Alt+D toggles dark mode, Alt+H goes home

🎯 Why This Matters

This isn't just about cleaner code (though that's nice). It's about creating a foundation that can scale. No more copy-pasting styles between files. No more inconsistent user experience. One change, everywhere it needs to be.

The dark mode implementation is particularly interesting—it respects your system preferences but lets you override manually. Your choice persists across page visits using localStorage.

🚀 The Technical Stack

  • DRY Principle: Don't Repeat Yourself—shared CSS/JS eliminates duplication
  • Progressive Enhancement: Works without JavaScript, better with it
  • Mobile-First: Responsive design that works on any screen size
  • Accessibility: Keyboard shortcuts and semantic HTML

🔍 Behind the Scenes

This update involved touching every HTML file, consolidating hundreds of lines of CSS, and implementing a state management system for theme switching. The battleship game alone required careful CSS overrides to maintain its unique styling while integrating with the global dark mode.

Working with GitHub Copilot on this refactor was like pair programming with someone who never gets tired of explaining CSS specificity rules. 🤖

✨ Try It Out

Click the 🔄 button to test dark mode, or try the keyboard shortcuts. The changes are live across the entire site. Sometimes the best updates are the ones you barely notice—everything just works better.
